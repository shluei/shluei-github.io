#     JAVA专周开发实战
 #### 简介:基于JAVA的Bank项目开发实战,用于学生学习，学会 继承,多态等方法功能的实现。

### 一、 安装
1. **安装IDEA**
2. **安装必要插件**
 - JDK11
 - GitHub
### 二、 功能
- 用户账户管理：创建账户（储蓄账户、支票账户）
- 存款：向指定账户存款
- 取款：从指定账户取款
- 转账：在账户之间转账

### 三、使用方法
1. 在IDEA中打开 WEEK18 该文件
2. 点击你想测试的任意类
3. 运行它，等待，然后查看结果
